# CodeAlpha Banking System

## 📌 Project Description

This project is developed as part of the **CodeAlpha C++ Programming Internship**.

The application simulates a basic banking system using Object-Oriented Programming concepts.

## 🚀 Features

- Customer Management
- Deposit Money
- Withdraw Money
- Transfer Funds
- Transaction History
- Account Details

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming
- STL Vector

## ▶️ How to Run

Compile:

```bash
g++ BankingSystem.cpp -o bank
```

Run:

```bash
./bank
```
